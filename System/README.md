250929 生物系统学
生物系统学？是关于生物物种多样性及物种之间相互关系的科学，它系统地研究生物、生物的层次系统，其研究结果是生物的分类系统，它是研究生物学其它问题的基础，它既古老又年轻，新的概念、观点和方法层出不穷。
  - 物种及物类的鉴定与分类(kinds and diversity)
  - 系统发育关系(phylogenetic relationships)
  - 地理分布(geographic distribution)
