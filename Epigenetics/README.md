什么是DNA甲基化（methylation）以及DNA甲基化的功能？
- 基因转录的沉默
  - 启动子被甲基化修饰后阻碍转录因子的结合，进而阻碍转录
  - DNA甲基化后招募Methyl-CpG-binding domain proteins(MBDs)结合，进而占据转录因子结合位点
  - MBD还可以进而招募其它蛋白(组蛋白等)
- 保护：稳定基因组避免被转座
- 基因组印记(imprinting): 父母来源遗传信息的表观信息不一致导致
- X染色体失活
- chromatin compaction 染色体压缩
- suppression of homologous recombination between repeats
- genome defense
