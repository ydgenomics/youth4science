# Review of increased knowledge of Python

九月中旬来到北京，转眼之间已到12月，又到了冬季，北京的冬季比重庆的冬季天气更明媚。Python作为第一个即将结课的课程，我期望能够得到一个好的分数，以足够的准备去面对期末。Python我本身学了很长一段时间了，从大四到现在，python无时无刻不被使用，我自信自己对python有一定的了解，但是想要完全脱离AI去开展一个项目实在很难，本着对自我要求可以独立敲代码的要求，更加系统学习一门编程语言，所以选择了这门选修课。

什么是编程，什么是python编程？比其它语言更容易学习且具有很好的社区的一门编程语言，如果要推荐学习的话，那一定是先学python。
编程基础？编程流程；程序的组织(输入-处理-输出)；
基础语法？变量和常量；数据类型；运算符和表达式；赋值语句
流程控制？条件(判断)；循环
模块与函数？包-模块-函数
数据结构？序列(字符串，列表，字典，集合，元组)
文件处理？文件类别(txt,csv,pickle);操作(读取，写入，复制，删除)
面向对象编程？对象；属性和方法；继承
异常处理与调试？异常的定义与分类；异常的处理；异常的测试与调试
